# video-transcode-tools
Video transcoding tools and scripts

Скрипт обрабатывает все видеофайлы в текущей директории, производится нормализация звуковой дорожки.

Необходимые инструменты (<a href="https://github.com/vearutop/video-transcode-tools/blob/master/windows-video-transcode-tools.7z?raw=true">transcode.7z</a> 8 MB) можно сделать доступными в PATH.

Сильное сжатие 480 x 320, общий битрейт около 100-250 кбит/с: transcode.cmd

Среднее сжатие 640x480, общий битрейт около 300-400 кбит/с: transcode-normal.cmd

Конвертирование в 3GP для античных мобильных устройств, 320x240, битрейт около 200 кбит/с: transcode-3gp.cmd
